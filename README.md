Best project structure for your React App
-----------------------------------------
> the idea of this repo is to show how you can structure files by using folders
```
react-folder-structure
├── build
├── node_modules (.gitignore)
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
├── src
│   ├── components
│   │   ├── app
│   │   │   ├── app.css
│   │   │   ├── app.js
│   │   │   └── app.test.js
│   │   └── index.js
│   ├── images
│   │   └── logo.svg
│   ├── utils
│   │   ├── ...
│   │   └── index.js
│   ├── index.css
│   ├── index.js
│   └── service-worker.js
├── .gitignore
├── package.json
└── README.md
```
